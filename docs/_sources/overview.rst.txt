Overview
========

This package implements a ROS-based navigation system with two main components:

1. **Action Client Node** - Interactive goal management
2. **Service Node** - Last goal tracking

Key Features
------------

- Interactive terminal interface for goal management
- Real-time robot position and velocity publishing
- Persistent last goal tracking via ROS service
- Integrated with Gazebo simulation environment
